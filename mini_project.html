Super Super Super Mini Project - Human Benchmark Typing Bot

This is a typing bot of Human Benchmark - Typing. The bot was written in Python, and it makes use of the 3 libraries listed below:

  1. Selenium
  2. Pyautogui
  3. Time



Video Demo





Source Code

                        from selenium import webdriver
                        import pyautogui
                        import time


                        def main():
                            driverPath = "/Users/michaelng/Desktop/1/Code/Python/chromedriver"  # Replace the path of driver on your own computer
                            driver = webdriver.Chrome(driverPath)
                            driver.get("https://humanbenchmark.com/tests/typing")
                            driver.maximize_window()  # Maximize the window

                            characters = driver.find_element_by_class_name("letters")
                            passage = characters.text

                            pyautogui.click(715, 535)
                            pyautogui.typewrite(passage)

                            time.sleep(5)
                            driver.quit()


                        main()
